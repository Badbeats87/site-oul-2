Welcome to the Component Style Guide!

This guide documents all reusable UI components used in the Vinyl Catalog admin dashboard. Each component includes interactive examples, usage guidelines, accessibility information, and copy-paste code examples.

Built on the Design System (see backend/styles/design-system.css) with 100+ design tokens and 200+ utility classes for consistent, maintainable frontend development.

Design System
